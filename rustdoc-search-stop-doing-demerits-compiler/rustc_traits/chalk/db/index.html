Provides the RustIrDatabase implementation for chalk-solve

The purpose of the chalk_solve::RustIrDatabase is to get data about specific types, such as bounds, where clauses, or fields. This file contains the minimal logic to assemble the types for chalk-solve by calling out to either the TyCtxt (for information about types) or crate::chalk::lowering (to lower rustc types into Chalk types).
