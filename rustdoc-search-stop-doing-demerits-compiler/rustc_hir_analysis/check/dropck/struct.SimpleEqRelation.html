    tcx: TyCtxt<'tcx>,
    param_env: ParamEnv<'tcx>,
}
Expand description

This is an implementation of the TypeRelation trait with the aim of simply comparing for equality (without side-effects).

It is not intended to be used anywhere else other than here.
